A simple Android app that displays a list of servers. Remember, this is just a test project!

### Features
- Login page
- Loading page
- List of servers

### Tech stack
- MVVM
- Clean architecture
- Kotlin
- Jetpack Compose
- Coroutines
- Retrofit
- Room

### Build
- Open up the project in Android Studio
- Wait for the project to sync
- Launch the app
